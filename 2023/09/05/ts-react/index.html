JSX 语法

JSX 是 React 库引入的一种语法，可以在 JavaScript 脚本中直接书写 HTML 风格的标签。

TypeScript 支持 JSX 语法，但是必须将脚本后缀名改成.tsx。

.tsx文件中，类型断言一律使用as形式，因为尖括号的写法会与 JSX 冲突。
